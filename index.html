Making pizza is similar to coding. There are certain things we have to do first. like making the base, letting it rise, mould the base out, add the sauce, add the toppings, add the cheese, bake, and cut and serve. Not all pizzas end up the same, but they do stsart out in the same way as long as you follow the order correctly. You can have everyything you need to make different pizzas using the same base structure that is set up before hand.

  1. make the pizza base
  2. add the comato sauce
  3. add the yummy toppings
  4. add the cheese
  5. bake
  6. cut
  7. serve
